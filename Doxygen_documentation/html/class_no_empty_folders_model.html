bool NoEmptyFoldersModel::hasChildren ( const QModelIndex &  parent) const
override

Checks if the provided parent index in the file system model has children (folders with content).

This function returns true if the parent item is not set as never having children and if the directory iterator indicates the presence of items. It's utilized to eliminate unnecessary folder expansion options.

Parameters
parentThe model index of the parent item to examine.
Returns
True if the parent item contains children (non-empty folders); otherwise, false.
